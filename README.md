# GSE 301 – Dataset Management and Basic Analysis System

This project is part of the GSE 301 Data Science course.

The program reads numerical and categorical data from text files,
calculates basic statistics such as total, average, minimum, and maximum,
and saves the results into a report file.

## Files Included
- dataset_analysis.py: Python program
- data_numbers.txt: Numeric dataset
- data_categories.txt: Category dataset
- analysis_report.txt: Output report

## How to Run
Run the Python file using:
python dataset_analysis.py
